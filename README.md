# 📺 Devflix

## 📖 Descrição do Projeto

Devflix é uma aplicação de console desenvolvida em Java, baseada nos conceitos de Programação Orientada a Objetos (POO) e Java avançado. O objetivo do projeto é criar uma plataforma simples para gerenciar filmes e séries, com funcionalidades específicas para administradores e clientes. Utilizamos JSON como mock para simular o armazenamento de dados, facilitando testes e aumentando a eficiência na usabilidade.

## ✨ Funcionalidades Principais

- **🎬 Cadastro de Filmes e Séries**: Administradores podem adicionar, editar e remover filmes e séries da biblioteca.
- **📜 Listagem de Filmes e Séries**: Tanto administradores quanto clientes podem visualizar a lista completa de filmes e séries disponíveis.
- **▶️ Assistir Filmes e Séries**: Clientes e administradores podem simular que assistiram aos filmes e séries cadastrados.
- **⭐ Avaliação de Filmes e Séries**: Ao assistir um filme ou série, os usuários podem avaliá-los numa escala de 0 a 5 estrelas.
- **🏆 Ranking**: Geração de um ranking dos filmes e séries mais assistidos durante a execução.

## 🗂 Estrutura de Dados

Utilizamos JSON como mock para simular o armazenamento de dados. Este formato inclui informações básicas sobre filmes, séries e usuários.

## 🏗 Arquitetura

A aplicação segue o padrão de arquitetura MVC (Model-View-Controller), garantindo uma estrutura organizada e de fácil manutenção.

## 📌 Conclusão

O projeto Devflix aplicou conceitos avançados de Java e Programação Orientada a Objetos (POO), resultando em funcionalidades essenciais para uma plataforma de streaming em console. A utilização do JSON como mock e a arquitetura MVC garantiram uma estrutura organizada e de fácil manutenção.
